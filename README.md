![projectImage](https://github.com/user-attachments/assets/71876ad4-a866-4f0b-a599-872230278ace)


## 💬 Service Description
```
결제 데이터를 기반으로 선정되는 그날의 히든글자를 바탕으로
십자말, 빙고, 예측 게임을 할 수 있어요!
혼자서, 팀원들과 협력하며 포인트를 모을 수 있어요 :)
```
